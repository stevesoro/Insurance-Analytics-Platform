## Overview

### Fictional Business Scenario

This project simulates a real-world assignment in which a newly hired Data Engineer at an insurance company is tasked with designing and implementing a production-ready analytics platform.  The objective is to build an end-to-end data engineering solution that ingests raw insurance data, applies data quality and transformation processes, organizes data using a Medallion Architecture (Bronze, Silver, Gold), and delivers curated datasets for business intelligence and reporting in Power BI.  Throughout the project, industry best practices are applied for data engineering, dimensional modeling, version control, documentation, and analytics development.
